## 为何还保留户籍

户籍我记得现在只有三个国家,中国,朝鲜,非洲的一个国家.不管从什么方面,户籍制度都是弊远大于利的,有兴趣可以看下 <向心城市>这本书,如果想让中国劳动力充分发挥价值,必须要让人员能流动起来



为啥一线城市的房价这么高,除了思想开放导致有很多有钱人以外.更多的是教育和医疗资源远不是小地方能比的

而你想充分享受一个地方的教育和医疗,你必须要有那个地方的户口,而你想真正落户,落到个人名下,而不是集体户口,你又必须要买那个地方的房子

所以逻辑变成了 有房子 --> 你能落户 --> 你才能享受本地的教育和医疗.你买的实际不是房子,而是教育医疗资源



而医疗和教育里面,教育的支出,是远大于医疗的,而且现在是必需品.中国的教育支出,不说第一,也是前几的.再苦不能苦孩子,实际是对于大部分人是,再苦也要让你享受教育资源,为啥这么热衷教育资源,除了东亚的文化,还有一点是教育的极度不均衡.(这个放到第二点讲)

所以一旦取消了户籍这种东西,他的逻辑就玩不转了,大家就不去买房了,他房地产这个支柱会完全崩塌.再也无法吹 GDP 高速增长

而为了让这套逻辑走下去,他必须把这个前提限制的死死的,并且要人为制造不均衡或者是特权的存在,因为这样你才会高价去买这个资源. 这个不均衡就体现在下面一点高考制度上

## 高考真的平等吗

教育资源我觉得分两个方面,一个是师资力量.另一个是好大学的招生比例

所以就会有四种:

师资力量强,招生比例高(北上)

师资力量强,招生比例不高(河南,山东,江浙)

师资力量不强,招生比例高(天津,这个可能不是特别恰当)

师资力量不强,比例不高(新疆,西藏)

其实大部分人看中的教育资源不是前者,而是后者.即我到底好不好考好大学.所以都会优先选北上,然后天津(十几年前,现在天津不太行了),最后才是内地师资力量好的上学,最后回到新疆等地考试,毕竟降维打击.比如前几天河南的跑去西安考试



一提到高考,就是中国人多,没法采用西方的制度,只能采用现在的方式,简单粗暴的选拔.

实际上这个有两方面,客观上中央集权,他是没办法管的了这么多人,又要考虑各种因素.他只能用这种简单粗暴的方式,实际上是懒政,也是无能为力;另一方面主观上,他本身就是想制造这种不均衡,好贩卖教育资源,以此炒高房价,拉动 GDP.只是他估计自己也没想到能炒这么高

都知道北上户口好考大学, 985/211 录取率是其他偏远地区省份的几倍

北上享受着最好的教育资源(师资方面),招生比例又高,导致成简单模式,因为户口把你限制死了,其他地方的根本没可能去北京高考,在一个市场根本不可能存在竞争的情况下就会出现这种.北上简单模式,河南山东江浙地狱模式

而他如果想让高考公平一点,那就把户籍取消,谁都可以去北上考试.这样子,地狱模式的人都会寻租跑去简单的地方,简单的地区就会往地狱发展,地狱地方的就会变简单.所以哪怕是按照现在的招生比例,虽然师资力量地区上不均衡,也不可能均衡,全国考好大学的难度最终也会趋于一致.

当然上面的是理想模型(不考虑回流生,即跑一线上学,回新疆考试这种),如果他权力不透明,信息不透明,信息不公开,他是没法达成让市场调节的.所以这就是为啥他不能也不想让高考公平



## 为何房价能这么高

房价能这么高,有以下几个原因

1. 绑定了户籍,教育/医疗.成了所谓的必需品
2. 普通人唯一增值的工具.
3. 政府财政,最终受益人

### 1. 必需品

买房是必需品吗? 肯定不是,政府为了贩卖资源,让你以为是必需品,而且通过教育,医疗等手段,强制让你买

结婚要房子

上学要房子

养老要 房子

不是必需品也得成必需品,所以之前彩礼要房子车子票子,政府压根 不想管甚至乐见其成,当人不结婚了,才出来说两句,打击高彩礼,但是却没对房地产做啥动作

### 2. 增值工具

普通人能赚钱/理财的东西一般只有两个,股票/基金和房地产,也可以加上银行存款,(但是收益率太低,跑不赢通胀)

而在中国金融外汇严格管控和信息极其不透明下,大 A 股就是割韭菜.但是你又买不了其他国家的股票.不想当韭菜把钱只能转向房地产

所以每当国家放水,钱都是流向房地产,因为19 年前收益那可是随便一年 10%,核心地方比如深圳哪怕是20年左右半岛城邦半年都涨了 30%.更不要说更早之前一年就能翻倍的情况.

这样的高收益情况下,谁还不去买房投资.毕竟一年赚的钱,你打一辈子工都赚不到

然后就变成击鼓传花的游戏,只要有人认为他还能涨,那就能一直有人接盘下去

## 政府想让房价降低吗

不可能

# 19. 关于感冒以及流感 20221228

感冒(国内一般都认为普通感冒，本篇中专指普通感冒)

不知道从啥时候起，国内出来了这种论调。"新冠是小感冒"。然后很多人信以为真，得了之后发文，"谁跟我说的新冠是小感冒？"，如果你或者身边人接触到的都是这个论调，那你获取到的99%都是垃圾信息。你可能就是个韭菜，割完一茬又一茬

大号流感对于目前奥密克戎版本来说是最准确的。然后关于信息是这样传递失真的 奥密克戎是大号流感 -> 奥密克戎是小流感 -> 奥密克戎是小感冒

感冒和流感症状很相似，伴随发烧，流涕，头晕，鼻塞，咳嗽，乏力等，虽然症状相似但是不是一样的东西，简单比喻，流感是老虎，感冒是小猫，虽然都是猫科，长得也像，你不能说两个危险程度一样吧

感冒一般来说都是细菌感染，流感都是病毒感染，同时因为抵抗力下降和炎症，伴随细菌感染。所以流感症状基本是比感冒重很多的

因为两个症状很相似，导致医生只看你症状听你描述是搞不清楚到底是什么病，所以你感冒了去医院，医院都让你做血常规，因为只有验了血才能知道你是细菌感染还是病毒感染。(这也是我一直说中医治不了病的根本原因，他连病因都不知道，怎么给你治病呢？头痛医脚)

不过如果你已经发烧了，比较紧急的时候，在血常规还没出来之前，会先给你开一点点消炎药(抗生素),抗生素只对细菌有效，对病毒一点用没有。但是不管普通感冒还是流感，都会伴随细菌感染，所以抗生素还是会对你的病起到一定作用。但是对于病毒感染一定要吃抗病毒的药物才行(比如我上次甲流开的奥司他韦，不是市面上你买到的什么叫抗病毒胶囊，三九感冒灵的，各种以抗病毒命名的中药，这种东西对于病毒一点用没有)

对于普通感冒，抗生素可能能起到90%的作用，因为根本原因是细菌，10%靠你免疫力；但是对于流感，抗生素可能只能起到10%的作用。剩下的靠你免疫力

>

>

其实没有中西医只说，只有现代和传统医学之分。以下中医指代传统医学，西医指代现代医学

为啥看个感冒要验血

感冒产生的原因一般分为两种，细菌性感染和病毒性感染。病毒性引起的一般会伴随细菌感染。比如我这次

以前去看感冒没有验血，只是不规范。说白了就是瞎几把整。不管你啥原因感冒都给你开一堆抗生素(你能药店买到的基本都是抗生素，头孢类，阿莫西林，左氧氟沙星等等)。而抗生素，对病毒性感染的根因——病毒是没有任何卵用的。但是你感觉有用，90%的原因是你的免疫系统，剩下的就是安慰剂作用，和小部分消灭了细菌，让你的症状看起来没那么严重，仅此而已。感冒了吃抗生素准有效，哪怕对病毒性感染效果不佳，所以每次感冒一般都会有抗生素药物

流感都是病毒感染引起的，没有例外。所以流感每次如果不去医院开正确的抗病毒的药物，只吃自己家里的抗生素，你只会感觉反反复复好不了，而且症状比一般感冒严重
